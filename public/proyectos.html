Ciao Ciao Pizza

Descripción
CiaoCiaoPizza es un proyecto web que hice en equipo, trata sobre una pizzeria en la provincia de Entre Rios. La pagina explica sobre la pizzeria, quien son ellos, los productos que tienen y añade un formulario de contacto.

Tecnologías:
Html - Css - JavaScript

Encriptador de texto

Descripción
Este un proyecto web que hice en solitario, tiene la finalidad de encriptar y desencriptar texto que el usuario ingrese. La pagina se divide en 2 secciones, una donde el usuario ingresa texto y otra donde el usuario puede visualizar el texto encriptado/desencriptado.

Tecnologías:
Html - Css - JavaScript
